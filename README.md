# SmartSales Forecast 

**AI-Powered Sales Dashboard for Small Businesses**

SmartSales Forecast is a **modern, interactive SaaS dashboard** built to help businesses visualize their sales data and make data-driven decisions effortlessly.

## Features

Upload sales data (CSV) seamlessly
View **7-day AI-powered sales forecasts**
Track **KPIs, trends, and inventory alerts**
Interactive charts with smooth animations
Fully responsive and supports **light/dark mode**

## Tech Stack

**Frontend:** React, TailwindCSS, React Router
**Charts & Visualization:** Chart.js, react-chartjs-2
**Future-ready:** Backend integration for AI forecasting
