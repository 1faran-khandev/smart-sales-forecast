## SmartSales Forecast

AI-Powered Sales Dashboard for Small Businesses

SmartSales Forecast is a modern web dashboard that helps businesses upload sales data, visualize trends, and see 7-day forecasts. It’s easy to use, responsive, and works in light/dark mode.

## Features

Upload CSV files and preview your data

View 7-day sales forecasts

Track KPIs and trends

Interactive charts with animations

Fully responsive and dark/light mode support

##  Tech Stack

React for frontend

TailwindCSS for styling

React Router for navigation

Chart.js and react-chartjs-2 for charts

Recharts for forecast charts

PapaParse for CSV parsing

##  Getting Started
1. Clone the repository
git clone https://github.com/<your-username>/smart-sales.git
cd smart-sales

2. Install dependencies
npm install

3. Run the app
npm run dev


 Open http://localhost:5173
in your browser.
